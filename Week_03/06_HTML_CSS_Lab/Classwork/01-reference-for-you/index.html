Custom Fonts


This is Arial, a system font. It's on every computer that accesses the internet.
This is Galada, a google font display font. It's not on any computer, so Google provides it for you - so nice of them.
This is League Gothic, a popular condensed custom font used for headlines quite often. It's not on any computer either and google doesn't serve it up either. What can you do? You can serve that custom font yourself with particular @font-face declarations (check the code).
